PAINT Example Implementation

This web application is an example of the implementation of PAINT within the PHP5 environment.

Before you start:

Before you attempt to run any parts of this example, you will need to update your credentials as follows:
  • Open PaintSession.cs which is in "App_code\com\pure360\paint"
  • Navigate to the constructor method "PaintSession()"
  • Replace the strings "yourUsername" and "yourPassword" with your own credentials
  • Save and rebuild the website
Example operations:

List Upload: Upload a list of data held within a text field.

Search messages: Search messages in the profile.

Search emails: Search emails in the profile.

Search deliveries: Search deliveries in the profile.

Create Message and Send : Create your message content and schedule an immediate delivery.

Create an email : Create your email content .

View an existing delivery : Load high level report data for an existing delivery

Send a One-to-One : Send a one-to-one tracked email

Send multiple One-to-Ones from multiple login sessions

Retrieve event notifications : Retrieve notifications of events such as clicks and opens
